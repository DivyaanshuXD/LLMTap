// core capabilities
Built for engineers
who ship AI agents.
🔍
Transparent ES Proxy instrumentation
Wraps any LLM client — OpenAI, Anthropic, Gemini — with a single function call. Your client behaves 100% identically. Streaming, tool calls, multi-turn conversations, function calling. All traced automatically, zero behavior changes, zero refactoring.
wrap(client) — that's it
⚡
Real-time SSE dashboard
Traces stream into the browser via Server-Sent Events the instant they complete. Token counts, latency, full message content — all live.
💰
Per-call cost intelligence
Built-in pricing for 50+ models. See exactly which pipeline step burned $0.40, not just the monthly aggregate.
🗺️
Agent trace grouping
Group multi-step pipelines into a single trace with startTrace(). Total cost and token budget at a glance.
🔒
Privacy by default
SQLite on your machine. Your prompts never leave localhost. OTLP export to Datadog, Grafana, or Jaeger when you need it.
📡
OpenTelemetry native
Follows GenAI Semantic Conventions. Drop-in OTLP forwarding to any production observability backend.

import OpenAI from "openai";
import { wrap } from "@llmtap/sdk";

const client = wrap(new OpenAI());

// open localhost:4781
// every call traced in real time
// costs · latency · full content
// your agent is now readable

const res = await client.chat
  .completions.create({...});
